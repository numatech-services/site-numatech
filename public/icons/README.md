# Icons PWA

Remplacer ces placeholders par vos vrais icons :

1. **icon-192.png** (192×192px) - Icon PWA petite taille
2. **icon-512.png** (512×512px) - Icon PWA grande taille  
3. **apple-touch-icon.png** (180×180px) - Icon iOS

## Génération à partir de votre logo

```bash
# Installer ImageMagick
apt-get install imagemagick

# Générer les icons
bash scripts/generate-pwa-icons.sh
```
